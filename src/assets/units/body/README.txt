GIMP
спрайт состоит из 4 частей
bottom - ходовая часть
название тела - основная часть тела
mask - маска где весь черный заменен на прозрачность (цвет в альфа канал) + (порог прозрачности 0.300) 
mask2- маска где весь белый заменен на прозрачность (цвет в альфа канал) + (порог прозрачности 0.500) +  уровни -> корекция световых уровней -> уровень на выходе 255,255

папка skins - заготовленные раскрашеные спрайты без масок
